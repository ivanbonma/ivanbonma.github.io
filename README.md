## 📝 Contenido

Este blog incluye:

- Tutoriales de desarrollo
- Guías prácticas
- Reflexiones sobre tecnología
- Documentación de proyectos

## 🛠️ Tecnologías

- **GitHub Pages** - Hosting
- **Jekyll** - Generador de sitios estáticos
- **Markdown** - Formato de contenido
- **HTML/CSS** - Estilos personalizados

## 📂 Estructura

```
.
├── _config.yml          # Configuración de Jekyll
├── _layouts/            # Plantillas HTML
├── _posts/              # Artículos del blog
├── index.md             # Página de inicio
├── about.md             # Página sobre mí
├── blog.md              # Página de blog
├── Gemfile              # Dependencias Ruby
└── README.md            # Este archivo
```

## 🚀 Cómo ejecutar localmente

```bash
# Instalar dependencias
bundle install

# Ejecutar servidor local
bundle exec jekyll serve

# Visitar en el navegador
http://localhost:4000
```
